# 🚗 Car Price Prediction with Machine Learning

## 📌 Project Overview

This project predicts the price of a car using Machine Learning techniques. The model is trained on car-related features such as brand, year, mileage, fuel type, horsepower, and transmission type.

The goal is to build a regression model that can estimate the selling price of a car based on its specifications.

---

## 🎯 Objective

* Analyze car dataset features.
* Perform data preprocessing and cleaning.
* Train a Machine Learning regression model.
* Predict car prices accurately.
* Evaluate model performance using regression metrics.

---

## 🛠 Technologies Used

## 📂 Dataset Features

The dataset may include:

* Selling Price

## ⚙️ Machine Learning Workflow

2. Data Cleaning
3. Feature Engineering

7. Model Evaluation

## 📊 Model Used

---

## 📈 Evaluation Metrics

* R² Score

## ▶️ How to Run

1. Clone the repository.
2. Install required libraries:

```bash
pip install pandas numpy matplotlib scikit-learn
```

3. Run the Python file:

```bash
python car_price_prediction.py
```
